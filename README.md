### 설치
```js
$ pnpm vite create // react typescript swc, 경로 ./
$ pnpm i
$ pnpx storybook@latest init
```

### 실행
```js
$ pnpm run dev /* 서버 */
$ pnpm run storybook /* 스토리북 */
$ pnpm run deploy /* 배포 (호스팅) */
$ pnpm run chromatic /* 배포 (스토리북) */
```
